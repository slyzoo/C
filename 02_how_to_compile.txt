how to compile and run a C program using cmd on windows

predoings : 
    make sure your file uses underscores / dashes instead of spaces
    make sure you type in the whole file name

step 1 : 
    open cmd
    type in "cd {file path to the C files}" and press enter

step 2 :
    type in "gcc {name of the file}.c"
    it will create a file named "a.exe"

step 3 : 
    type "a.exe" in cmd
    
    boom you're done :3
